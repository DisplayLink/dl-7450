:mod:`iperf` --- iperf3 network performance testing
=============================================================

.. module:: iperf
  :synopsis: iperf3 network performance testing

This module contains classes for running iperf3 network performance tests.
These classes are primarily designed for manufacturing tests, where network
performance testing is most pertinent, but are also provided for
application developers to use as they wish.
